# 📚 StudyAI – Smart AI Study Companion & Quiz Generator

StudyAI is a full-stack AI-powered learning platform developed as part of the **APSCHE Internship Programme**.

The application allows students to upload study materials and use Generative AI to create summaries, flashcards, practice quizzes, personalized study plans, and analyze learning performance.

## 🛠️ Technology Stack

**Frontend:** React.js, JavaScript, Axios, Chart.js

**Backend:** Python, Flask, REST APIs

**Generative AI:** Groq API, Llama 3.3 70B Versatile

**Database:** Firebase Firestore with Local JSON Storage Fallback

**Deployment:** Render
